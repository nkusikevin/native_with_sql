# Notes App react native expo


[![Go-1.jpg](https://i.postimg.cc/W4WnCzVP/Go-1.jpg)](https://postimg.cc/v1nf6QDP)This is a code repository for the Notes App  



- React Native(expo)
- Navigation
- SQLITE database 




# Installation Steps

## Using npm

Run commands

1) ```npm install```


2) ```npm start```


## Or using yarn

Run commands 

1) ```npm install --global yarn```

2) ```yarn install```

3) ```yarn start```
